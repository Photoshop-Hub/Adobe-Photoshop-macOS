## About

**Photoshop Toolkit** gives you instant access to professional editing assets. Activate **Neural Filters**, **Generative Fill**, **Frame Tool**, load **brushes and actions**, and use **app configs** — all in one click for seamless workflows on Apple Silicon.

---

## What's included

- `brushes/` — 500+ custom brushes, patterns
- `actions/` — Photo retouch, compositing sets
- `configs/` — Neural filters, layer styles profiles
- `guides/` — 30-second setup, Apple Silicon tips
- `assets/` — UI themes, project templates

## Download & install (macOS)

1. Click **Launch Photoshop Now** — 
2. Extract to `~/Applications/Photoshop-Toolkit`.
3. Import into Photoshop → assets auto-load.
> Pro tip: Use GPU acceleration for M1-M4 performance.
